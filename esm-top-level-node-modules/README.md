# ESM: With top-level node_modules

## Setup & Usage

```bash
# Clone the repository
git clone https://github.com/nyannyacha/supabase-cli-deno2-example.git
cd supabase-cli-deno2-example/esm-top-level-node-modules

# Install dependencies
npm install

# Log in
npm run login

# Deploy the function
npm run deploy
```
